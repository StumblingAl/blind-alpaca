Explanation

Set Up

I've used a clone of angular-seed to set this project up. Therefore the backend runs on NodeJS http-server. Both NPM and Bower have been used to get the project running and install the various dependencies.

Home (View 1) uses the $http/ngRoute dependencies to demonstrate use of a public REST API. A POST request is sent to the API with the parameters taken from the user input.

MVC

  • View: 3 of these have been included - 1 per application menu item
  • Controller: 1 controller per view, split into functions

AngularJS v1.5

  • $http get and post examples - ngroute
  • Inclusion of external libraries - angular-spinner
  • Error handling

Git

Clone of Angular-Seed project. Then forked to blind-alpaca (replacing the origin).

Bootstrap

Bootstrap 3 has been used, utilising the grid layout included with it and other standard optimised properties.
